结合Svelte 5的Runes特性，利用Skeleton UI库，构建一个类似于外卖点单的页面。关键信息：
1. 页面分为左右两栏，左侧大概占屏幕宽度的20%，右侧占80%。
2. 左右两侧都是一个单列表。
3. 左侧列表为灰色底纹，从上到下排列各种商品的分类；右侧列表无底纹，从上到下排列各种商品，每种商品分类前都有一个栏目为商品分类名称。
4. 点击左侧某个分类，右侧列表会自动跳转到指定商品分类。
5. 补充第4点内容：不用考虑右侧列表中没有对应分类而无法跳转的情况，因为模拟数据的结构如下，左侧分类直接就是从数据中提取出来的：
```json
{
    "Category1": [
        { "id": "1", "name": "name1" },
        { "id": "2", "name": "name2" }
    ],
    "Category2": [
        { "id": "3", "name": "name3" },
        { "id": "4", "name": "name4" }
    ]
}
```
